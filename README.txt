cookie consent banner for non-drupal sites
modified from: https://www.websitepolicies.com/create/cookie-consent-banner


	1. upload cookie-consent.css and cookie-consent.js to web server

	2. edit paths in cookie-includes-head.html and cookie-includes-foot.html to where you uploaded cookie-consent.css and cookie-consent.js

	3. copy COMPLETE contents of edited cookie-includes-head.html into the <head>...</head> section and cookie-includes-foot.html before closing </body> tag of all pages to display banner (or into corresponding includes files)

	4. that's it, refresh page to see the banner
